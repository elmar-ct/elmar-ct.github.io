Continuous Maps

One can view a continuous map as just a homomorphism in the category Top of all topological spaces. Such definition of course does not provide much details for us to check in practical situations wether given maps are continuous. We have to descend to point-set topology to form thorough understanding of continuity. Then, we see that a continuous map is just a map between topological spaces, with the additional condition that it must premap any open onto an open. In some situations, this can consist of checking all the elements of the class topology for the codomain; in some case, only a much restricted class of open sets needs to be verified.

In order for \(f: X\to Y\) to be continuous, it is enough that the preimages of sets comprising some prebase of \(Y\) be open. Such is the case when the class topology on \(Y\) has bases and prebases.

Principle of continuity

$$ f: X\to Y \text{ is continuous }\iff \text{the preimage of any open is open}$$

Composition

If \( f: X\to Y\) and \(g: Y \to Z\) are continuous then \(g\circ f: X \to Z\) also is continuous. Indeed, for all open \(O \in \mathcal{O}(Z)\), \((g\circ f)^{-1}(O) = f^{-1}(g^{-1}(O)) \). But \(g^{-1}(O)\) is open in \(Y\) since \(g\) is continuous. Then \(f^{-1}(g^{-1}(O))\) is open in \(X\).

Modifications continuous

Let \(f: X\to Y\) be continous. For any \(A \subset X\) and \(B\subset Y\), if \(f(A) \subset B\), then the compression \(\operatorname{ab} f: A\to B\) is continuous. In particular, the restriction \(f\vert_A\) is continuous. Similarly, \(\operatorname{ab} f: X \to B\) is continous if and only if \(f: X\to Y\) is continuous. In particular, \(f: X\to Y\) is continuous if and only if \(f: X\to f(X)\) is continuous.

Continuity and fundamental covers

If \(\Gamma\) is a fundamental cover of \(X\), then \(f:X\to Y\) is continuous if and only if \( f\vert_{A'} : A' \to Y\) is continuous for any \(A' \in \Gamma.\) Equivalently, let \(\Gamma\) be a fundamental cover of \(X\), top space, and assume that for all \(A\in \Gamma\) there is a continuous map \(f_A: A\to Y\), such that \(f_A(x) = f_B(x) \) for all \(x\in A\cap B\) (\(A, B\in \Gamma\)); then the map \(f: X\to Y\) defined by $$ f(x) = f_A(x),\, \forall x\in A\, (A\in \Gamma) $$ is continuous.

A continuous map is open if the image of open sets are open; is closed if the image of closed sets are closed.

Sufficient condition for open map: a continuous map \(f: X\to Y\) is open if for each \(x\in X\), there is \(U_x\) neighborhood of \(f(x)\) and a continuous map \(g_x: U_x \to X\) such that the composition \( f\circ g_x \) coincide with \( in: U_x \to Y\).

Pointwise continuity

A map \(f: X\to Y\) is continuous at the point \( x\in X \) if for any neighborhood \(V\) of \(f(x)\) there is a neighborhood of \(x\) such that \(f(U) \subset V\). If the class topology of \(X\) has a prebase \(\Delta\) at the point \(x\) and if \(E\) is the prebase at \(f(x)\) then , \(f\) is continuous at \(x\) if and only if each neighborhood \(V \in E \) contains the image of some neighborhood \(U\in \Delta\).

If \((X, d_X)\) and \((Y, d_Y)\) are metric spaces; the map \(f: X\to Y\) is continuous at a point \(x\in X\) if for every \(\epsilon>0\) there is \(\delta >0\) such that \(d_X(x, x')<\delta\implies d_Y(f(x), f(x')) < \epsilon. \)

Homeomorphisms and Embeddings

An ivertible map \(f\) is an homeomorphism if both \(f\) and \(f^{-1}\) are continuous;

\(f:X\to Y\) is a topological embedding if \(\operatorname{ab} f: X\to f(X)\) is a homeomorphism.

Example of homeomorphism: the following mapping maps homeomorphically the open ball \(\operatorname{Int} D^n \) onto the euclidean space \(\mathbb{R}^n\) $$ f: x\mapsto \begin{cases} 2x\arctan( d(0, x)) / \pi d(0, x) & \text{if} x\neq 0\\ 0, & \text{if} x = 0\end{cases} $$

The inverse of \(f\) is given the map \(g: \mathbb{R}^n \to \operatorname{Int} D^n \) defined by: \( g: \mathbb{R}^n \to \operatorname{Int} D^n, \, g(\xi) = \dfrac{\xi}{d\left( 0, \xi\right)} \)
